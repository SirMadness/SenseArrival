# CI Reports

**Category**: `ci`
**Naming**: `ci-[branch]-YYYYMMDD-HHMM.md`

## What Goes Here

- CI pipeline execution results
- Build reports
- Lint check results
- Test run summaries
- Deployment pipeline results

## Example Reports

- `ci-main-20260202-1430.md` — CI run on main branch
- `ci-feature-auth-20260201-0930.md` — CI run on feature branch
- `ci-release-v2.0-20260131-1600.md` — Release pipeline results
