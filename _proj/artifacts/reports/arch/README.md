# Architecture Reports

**Category**: `arch`
**Naming**: `arch-[topic]-YYYYMMDD.md` or `ADR-XXXX-[topic].md`

## What Goes Here

- Architecture Decision Records (ADRs)
- System design documents
- High-level architecture diagrams
- Technology stack decisions
- Scalability and reliability designs
- Integration architecture

## Example Reports

- `arch-microservices-migration-20260202.md` — Plan for monolith to microservices
- `ADR-0001-database-choice.md` — Why we chose PostgreSQL
- `arch-api-gateway-20260201.md` — API gateway design
