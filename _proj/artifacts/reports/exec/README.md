# Execution Reports

**Category**: `exec`
**Naming**: `exec-[task]-YYYYMMDD.md`

## What Goes Here

- Command execution logs
- Script output summaries
- Deployment logs
- Migration execution reports
- Batch job results

## Example Reports

- `exec-db-migration-20260202.md` — Database migration execution log
- `exec-data-import-20260201.md` — Data import job results
