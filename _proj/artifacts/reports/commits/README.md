# Commit Reports

**Category**: `commits`
**Naming**: `commits-[scope]-YYYYMMDD.md`

## What Goes Here

- Commit summaries for releases
- Changelog generation
- Release notes drafts
- Git history analysis
- Merge summaries

## Example Reports

- `commits-v2.0-release-20260202.md` — Changelog for v2.0 release
- `commits-feature-auth-20260201.md` — Summary of auth feature commits
- `commits-weekly-20260131.md` — Weekly commit summary
